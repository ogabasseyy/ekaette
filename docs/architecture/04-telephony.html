2. Inbound Phone Call Flow (AT → SIP Bridge → Gemini)

sequenceDiagram
    participant CALLER as Caller (Phone)
    participant AT as Africa's Talking
(SIP Registrar)
    participant SIP as SIPServer
(GCE VM :6060)
    participant SESS as CallSession
(4-task pipeline)
    participant GEMINI as Gemini Live API
(v1alpha)
    participant FS as Firestore

    Note over CALLER,FS: SIP Registration (on startup)
    SIP->>AT: SIP REGISTER (UDP)
    AT->>SIP: 401 Unauthorized (nonce challenge)
    SIP->>SIP: Compute digest auth (MD5/SHA-256)
    SIP->>AT: SIP REGISTER (with Authorization)
    AT->>SIP: 200 OK (registered)
    Note over SIP,AT: Re-registers at 80% of expiry

    Note over CALLER,FS: Inbound Call
    CALLER->>AT: Dial 
    AT->>SIP: SIP INVITE (SDP: G.711, port X)
    SIP->>AT: 100 Trying
    SIP->>SIP: Allocate RTP port (10000-20000)
    SIP->>AT: 200 OK (SDP: G.711, port Y)
    AT->>SIP: ACK

    Note over SIP,FS: Session Setup
    SIP->>SESS: Create CallSession
    SESS->>FS: Write call start record
    SESS->>GEMINI: Connect Gemini Live
    GEMINI-->>SESS: WebSocket connected

    Note over SESS,GEMINI: Proactive Greeting
    SESS->>GEMINI: send_client_content("[Call connected]")
    GEMINI->>SESS: Audio greeting (PCM16 24kHz)
    SESS->>SESS: Encode 24k→8k→G.711→RTP
    SESS->>AT: RTP audio frames
    AT->>CALLER: Caller hears AI greeting

    Note over CALLER,GEMINI: Bidirectional Audio

    par Caller speaks
        CALLER->>AT: Voice audio
        AT->>SIP: RTP G.711 frames
        SESS->>SESS: G.711 decode→resample 8k→16k
        SESS->>GEMINI: send_realtime_input(PCM16 16kHz)
    and AI responds
        GEMINI->>SESS: PCM16 24kHz chunks
        SESS->>SESS: Resample 24k→8k→G.711→RTP
        SESS->>AT: RTP audio frames
        AT->>CALLER: Caller hears AI
    end

    Note over CALLER,FS: Call Ends
    AT->>SIP: SIP BYE
    SESS->>GEMINI: Close WebSocket
    SESS->>FS: Write call end record

3. WhatsApp Call Flow (Opus/SRTP → Gemini)

sequenceDiagram
    participant WA as WhatsApp Caller
    participant WA_SIP as WaSipClient
(GCE VM)
    participant WA_SESS as WaSession
(4-task pipeline)
    participant GEMINI as Gemini Live API
(v1alpha)
    participant FS as Firestore

    Note over WA,FS: WhatsApp Call Setup
    WA->>WA_SIP: WhatsApp call signaling
    WA_SIP->>WA_SIP: Negotiate Opus codec, SRTP keys
    WA_SIP->>WA_SESS: Create WaSession
    WA_SESS->>FS: Write call start record
    WA_SESS->>GEMINI: Connect Gemini Live (proactive_audio=True)

    Note over WA_SESS,GEMINI: Proactive Greeting
    WA_SESS->>GEMINI: send_client_content("[Call connected]")
    GEMINI->>WA_SESS: Audio greeting (PCM16 24kHz)

    Note over WA,GEMINI: Bidirectional Audio

    par Caller speaks
        WA->>WA_SIP: SRTP Opus frames
        WA_SESS->>WA_SESS: SRTP unprotect→Opus decode→PCM16 16kHz
        WA_SESS->>GEMINI: send_realtime_input(PCM16 16kHz)
    and AI responds
        GEMINI->>WA_SESS: PCM16 24kHz chunks
        WA_SESS->>WA_SESS: Opus encode→SRTP protect
        WA_SESS->>WA: SRTP Opus frames
    end

    Note over WA,FS: Call Ends
    WA_SESS->>GEMINI: Close WebSocket
    WA_SESS->>FS: Write call end record

4. SMS Text Bridge Flow

sequenceDiagram
    participant USER as User (Phone)
    participant AT as Africa's Talking
    participant CR as Cloud Run (FastAPI)
    participant GEMINI as Gemini (Text API)
    participant ANALYTICS as Campaign Analytics

    Note over USER,ANALYTICS: Inbound SMS → AI Response
    USER->>AT: SMS to virtual DID
    AT->>CR: POST /api/v1/at/sms/callback
    CR->>CR: Deduplicate by message ID
    CR->>GEMINI: Bridge text to Gemini
    GEMINI->>CR: AI text response
    CR->>CR: Truncate to 160 chars
    CR->>AT: providers.send_sms(reply)
    AT->>USER: SMS reply
    CR->>ANALYTICS: Record inbound reply event

    Note over USER,ANALYTICS: Outbound SMS Campaign
    CR->>AT: providers.send_sms(recipients[], message)
    AT->>USER: Bulk SMS delivery
    CR->>ANALYTICS: Record sent/delivered/failed
